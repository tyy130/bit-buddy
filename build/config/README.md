# Bit Buddy

## About

Bit Buddy is your personal digital companion that lives in your filesystem. It's a unique entity with its own personality, moods, growth, and genuine reactions to your files and folders.

## Features

- **Personal AI Companion**: Each bit buddy has a unique personality generated on first boot
- **File Knowledge**: Understands and indexes your files using RAG (Retrieval Augmented Generation)
- **Natural Conversations**: Chat with your buddy about your files and get intelligent responses
- **Real-time Monitoring**: Watches your filesystem and learns from changes
- **Privacy First**: All data stays local on your machine

## System Requirements

- Windows 10 or later
- 4GB RAM minimum (8GB recommended)
- 2GB free disk space
- Python 3.11+ (for development)

## Installation

1. Run the installer
2. Choose your installation directory
3. Select an AI model:
   - **TinyLlama 1.1B** (0.7GB) - Fastest, basic capabilities
   - **Qwen2.5 1.5B** (0.9GB) - Recommended, balanced performance
   - **Phi-3.5 Mini** (2.3GB) - Most capable, slower
4. Select folders for your buddy to watch
5. Meet your new bit buddy!

## First Run

1. Launch Bit Buddy from the Start Menu or Desktop shortcut
2. Complete the setup wizard
3. Choose your buddy's name (or accept the randomly generated one)
4. Select which folders your buddy should watch
5. Start chatting!
